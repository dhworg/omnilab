# OmniLab

!!! warning "Pre-alpha"

    image without ROS or Gazebo. Real docs land with Phase B.

See [`project-spec-v1.md`](https://github.com/dhworg/omnilab/blob/main/project-spec-v1.md)
in the repo for the full v1 plan.
