# {{project_name}}

ESP32 + micro-ROS firmware skeleton, generated by
`omnilab template install micro-ros-blink`.

## Build the firmware

```sh
omnilab up
omnilab micro-ros           # starts the agent inside the container
cd esp32
pio run                     # builds firmware/sim if no board attached
omnilab hw flash esp32 esp32/.pio/build/esp32dev/firmware.bin
```

Edit `esp32/src/main.cpp` to publish your own topics.
