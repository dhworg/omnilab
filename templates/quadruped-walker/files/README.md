# {{project_name}}

Quadruped + observers demo, generated by
`omnilab template install quadruped-walker`.

## Quick start

```sh
omnilab up
omnilab sim                 # quadruped in arena
omnilab observe --json      # live spatial state for an agent
omnilab observe --validate observers.yaml
```

Edit `observers.yaml` to add new motion classes or anomalies.
